# @facile-it/tsconfig

Shared TypeScript config for facile.it projects

## Installation

```sh
npm install -D @facile-it/tsconfig
```

## tsconfig common

Add to your `tsconfig.json`:

```json
"extends": "@facile-it/tsconfig/tsconfig.json"
```

## tsconfig Next.js

Add to your `tsconfig.json`:

```json
"extends": "@facile-it/tsconfig/tsconfig-next.json"
```
